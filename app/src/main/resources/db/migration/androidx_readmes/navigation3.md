# navigation3

Navigation 3 is a new navigation library designed to work with Compose. 

[[API Reference](https://developer.android.com/reference/kotlin/androidx/navigation3/runtime/package-summary)]

## Declaring dependencies

To add a dependency on navigation3, you must add the Google Maven repository to your project. Read [Google's Maven repository](https://developer.android.com/studio/build/dependencies#google-maven) for more information.

Add the dependencies for the artifacts you need in the `build.gradle` file for your app or module:

### Groovy
```
dependencies {
    implementation "androidx.navigation3:navigation3-runtime:1.1.0-alpha02"
    implementation "androidx.navigation3:navigation3-ui:1.1.0-alpha02"
}
```

### Kotlin
```
dependencies {
    implementation("androidx.navigation3:navigation3-runtime:1.1.0-alpha02")
    implementation("androidx.navigation3:navigation3-ui:1.1.0-alpha02")
}
```

For more information about dependencies, see Add build dependencies.
